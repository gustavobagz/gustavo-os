# Identidade visual — GSO Advogados

Logomarca oficial do escritório (conceito "contemporâneo — GSO ponto",
aprovado em 2026-07-16). Texto convertido em curvas nos SVGs — renderiza
idêntico em qualquer máquina, sem depender de fonte instalada.

## Paleta
| Uso | Cor | Hex |
|-----|-----|-----|
| Sigla (fundo claro) | Grafite | `#1C2127` |
| Ponto (fundo claro) | Bronze | `#A57B45` |
| Sigla (fundo escuro) | Marfim frio | `#EDEAE3` |
| Ponto (fundo escuro) | Ouro envelhecido | `#C29A63` |
| Fundo claro de referência | Papel | `#FAFAF8` |

Tipografia de origem: Segoe UI Black (sigla) + Segoe UI Semibold (ADVOGADOS).

## Arquivos
| Arquivo | Quando usar |
|---------|-------------|
| `logo-gso-principal.svg/.png` | Padrão — documentos, site, propostas (fundo claro; PNG transparente 1946×1094) |
| `logo-gso-fundo-escuro.svg/.png` | Sobre fundos escuros (slides, rodapé de site) |
| `logo-gso-mono-preto.svg` | Impressão P&B, carimbo, fax judicial |
| `logo-gso-mono-branco.svg` | Sobreposição em foto/fundo colorido escuro |
| `simbolo-gso.svg` | Símbolo isolado (G + ponto), transparente |
| `avatar-gso-escuro-1024.png` | Avatar WhatsApp/LinkedIn/Instagram (preferencial) |
| `avatar-gso-claro-1024.png` | Avatar em contextos que exigem fundo claro |

## Regras de uso
- Não esticar, não rotacionar, não trocar as cores fora da paleta.
- Área de respiro mínima: a altura do ponto bronze em todos os lados.
- Tamanho mínimo do lockup completo: ~120 px de largura; abaixo disso,
  usar só o símbolo (`simbolo-gso.svg`).
- Regeneração: script em versões futuras — pedir ao Claude Code, que
  reconstrói a partir das especificações acima.
