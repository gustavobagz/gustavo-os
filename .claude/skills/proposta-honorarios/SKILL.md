# Skill: Proposta de Honorários (Escritório Próprio)

## Quando usar
Gustavo pede uma proposta comercial para um cliente da prática independente
(fora da H2).

## Modelo de receita padrão
Misto: diagnóstico como porta de entrada, combinando projeto pontual com
recorrência (honorários fixos + êxito, conforme o caso).

## Fluxo
1. Coletar: nome do cliente, área (cobrança/execução, acordo de sócios,
   proteção patrimonial, vestibular, etc.), escopo do serviço, valor estimado.
2. Montar a proposta: diagnóstico inicial → escopo detalhado → modelo de
   honorários → prazos → condições.
3. Salvar em `Escritorio/Propostas/[cliente]/`.
